# 🌅 Eoynx 소개 - 비개발자를 위한 설명

## Eoynx가 뭔가요?

**Eoynx (이오닉스)**는 AI가 웹사이트를 더 잘 이해할 수 있도록 도와주는 플랫폼입니다.

### 쉬운 비유

**기존 웹사이트** 🏠
```
AI가 집에 들어왔을 때:
"어... 이게 뭐지? 버튼인가? 텍스트인가? 
가격은 어디 있지? 상품명은 어디?"
→ AI가 웹페이지를 이해하기 어려움
```

**Eoynx를 사용한 웹사이트** 🏠✨
```
AI가 집에 들어왔을 때:
"아! 여기가 상품 목록이고, 
이건 가격이고, 저건 버튼이구나!"
→ AI가 모든 것을 즉시 이해
```

---

## 왜 필요한가요?

### 문제점: AI가 웹사이트를 이해하기 어려움

| 상황 | 사람 👤 | AI 🤖 |
|------|---------|-------|
| "이 상품 가격이 얼마야?" | 바로 보임 | 어떤 숫자가 가격인지 모름 |
| "장바구니에 담아줘" | 버튼 클릭 | 어떤 버튼인지 모름 |
| "비슷한 상품 보여줘" | 스크롤해서 찾음 | 관련 상품 영역을 모름 |

### 해결: Eoynx가 AI에게 "지도"를 제공

```
"여기가 가격이야"  →  ₩29,900
"이게 장바구니 버튼이야"  →  [담기]
"이 영역이 관련 상품이야"  →  비슷한 상품 3개
```

---

## 어떻게 작동하나요?

### 1단계: 웹사이트가 Eoynx에 연결

```
웹사이트 주인: "우리 사이트에서 AI가 상품 검색하고 장바구니에 담을 수 있게 해주세요"
Eoynx: "네, 준비됐어요!"
```

### 2단계: AI가 Eoynx를 통해 접속

```
AI (예: ChatGPT): "이 사이트에서 노트북 검색해줘"
Eoynx: "여기 노트북 5개 있어요"

[노트북 A] ₩1,200,000 - 삼성
[노트북 B] ₩990,000 - LG
[노트북 C] ₩1,500,000 - Apple
...
```

### 3단계: AI가 행동 수행

```
사용자: "가장 싼 거 장바구니에 담아줘"
AI: "노트북 B를 장바구니에 담았어요!"
```

## 주요 기능

### 🔍 상품 검색
AI가 웹사이트의 상품을 검색할 수 있습니다.
```
"빨간색 운동화 찾아줘"
→ 빨간색 운동화 10개 결과
```

### 🛒 장바구니 관리
AI가 장바구니에 상품을 담고 확인할 수 있습니다.
```
"이거 장바구니에 담아줘"
→ 장바구니에 추가됨!
```

### 📦 주문 정보
AI가 배송 상태 등을 확인할 수 있습니다.
```
"내 주문 어디까지 왔어?"
→ 현재 배송 중, 내일 도착 예정
```

### 🌐 웹페이지 분석 (Edge Gateway)
AI가 다른 웹사이트의 정보를 가져올 수 있습니다.
```
"이 URL에서 상품 정보 뽑아줘"
→ 상품명: OOO, 가격: ₩XX,XXX
```

## 서버가 두 개인 이유

Eoynx는 두 가지 서버를 운영합니다:

### 🏠 메인 서버 (eoynx.com)
- **역할**: 쇼핑몰 기능 (검색, 장바구니, 주문)
- **보안**: 로그인 필요
- **비유**: 백화점 직원 👔
  - "회원이시군요, 뭘 도와드릴까요?"

### ⚡ 엣지 서버 (api.eoynx.com)
- **역할**: 웹페이지 분석
- **보안**: 누구나 사용 가능
- **비유**: 정보 안내소 🗺️
  - "이 주소의 정보를 알려드릴게요"

### 왜 분리했나요?

| 구분 | 메인 서버 | 엣지 서버 |
|------|----------|----------|
| 속도 | 보통 | 매우 빠름 (전 세계 300개 지점) |
| 기능 | 복잡한 쇼핑 기능 | 단순한 웹 분석 |
| 보안 | 로그인 필요 | 누구나 가능 |
| 위치 | 한국 | 전 세계 |

## MCP가 뭔가요?

**MCP (Model Context Protocol)**는 AI와 서비스가 대화하는 "언어"입니다.

### 비유: 번역기

```
사람 언어              MCP 프로토콜              컴퓨터 언어
"상품 검색해"   →   {"method":"search"}   →   데이터베이스 조회
```

AI가 다양한 서비스와 대화할 때 MCP라는 공통 언어를 사용합니다.

### MCP 도구란?

AI가 사용할 수 있는 "기능"들을 "도구"라고 부릅니다.

| 도구 이름 | 하는 일 | 비유 |
|-----------|--------|------|
| `search_products` | 상품 검색 | 🔍 돋보기 |
| `add_to_cart` | 장바구니 담기 | 🛒 카트 |
| `fetch_url` | 웹페이지 가져오기 | 📥 다운로드 |
| `extract_links` | 링크 추출 | 🔗 링크 수집기 |

## 자주 묻는 질문

### Q: 개인 정보는 안전한가요?
**A:** 네! Eoynx는 로그인 시스템을 사용하고, 허가된 AI만 접근할 수 있습니다.

### Q: 모든 웹사이트에서 작동하나요?
**A:** Edge Gateway(웹 분석)는 대부분의 사이트에서 작동합니다. 단, 일부 보안이 강한 사이트(은행, 쿠팡 등)는 접근이 제한될 수 있습니다.

### Q: 무료인가요?
**A:** 기본 기능은 무료입니다. 대량 사용 시 요금이 발생할 수 있습니다.

### Q: 어떤 AI와 호환되나요?
**A:** ChatGPT, Claude, Gemini 등 MCP를 지원하는 모든 AI와 호환됩니다.
